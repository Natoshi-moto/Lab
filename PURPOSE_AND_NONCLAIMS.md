# Purpose and non-claims

## Purpose

Nexus preserves project continuity outside any one model conversation, routes bounded context to replaceable AI seats, records what was inspected and executed, and keeps proposals, actions, evidence, interpretation, and canonical history mechanically distinct.

## Research purpose

The laboratory supports work across disparate fields without copying one object into multiple subject folders. Physical paths express lifecycle and authority; graph edges express subject membership and relationships.

## Governance purpose

Rules that change accepted state must be visible, versioned, reviewable, and enforceable by ordinary software. A router may coordinate work but may not grant itself authority.

## Non-claims

This repository does not claim:

- that Nexus is complete, secure, production-ready, or externally audited;
- that a Git commit, check run, receipt, or model agreement proves truth;
- that a private GitHub repository prevents authorized AI providers from processing routed content;
- that same-provider model agreement is independent corroboration;
- that current scaffolding is the final architecture;
- that `CANONICAL_AS_IS` means correct.

## Current maturity

R001 is an **audit-ready bootstrap candidate**. It demonstrates enforceable primitives and exposes its remaining assurance gaps.
