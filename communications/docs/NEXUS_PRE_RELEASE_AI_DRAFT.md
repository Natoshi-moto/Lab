## Non-negotiable boundary proposed by this draft

`SYNTHETIC DATA · NON-TRANSFERABLE · NON-REDEEMABLE · STRICT NO SALE`

The draft makes the software and user-grown creatures the intended source of utility. Tips, status, game energy and reputation are application records, not assets. It does not claim that open-source software can prevent off-platform agreements; it requires official builds and protocols to provide no sale, transfer, redemption, price, payout, wrap or bridge surface.

## Known contradictions requiring red-team review

The current tree still contains wallet, NEX balance, send, transfer, stake, mint and sats language, tests that expect some of it, raw-key browser storage in a legacy Nostr path, and incomplete Eidolin packaging. Nex-Sim material was located locally but has not been admitted into the canonical build. These are recorded as blockers rather than rewritten as completed compliance.
