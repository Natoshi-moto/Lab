# Failure pattern

A model with repository access may speak as though it inspected the complete corpus. Access, indexing, route selection, inclusion and actual inspection are distinct.

## Revival gate

Any design that relies on “the model can see the repository” must provide an explicit included-file manifest, unread-context declaration and participant inspection statement.
