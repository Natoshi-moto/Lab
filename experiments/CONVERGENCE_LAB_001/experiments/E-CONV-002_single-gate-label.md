# E-CONV-002 — Single-gate correlation labeling

**status:** `PROPOSED`  
**status_authority:** `NONE`

## Question

Does mandatory UI/docs label `CORRELATED_UNDER_ONE_GATE` reduce stranger misread of multi-seat work as multi-institution independence?

## Protocol (draft)

1. Show strangers (or seats role-playing strangers) two packets: unlabeled multi-seat PR stack vs labeled.  
2. Ask: “How many independent organizations signed this?”  
3. Compare error rates.

## Gate

Human decides if ethical/external subjects allowed; else seat-only pilot.
