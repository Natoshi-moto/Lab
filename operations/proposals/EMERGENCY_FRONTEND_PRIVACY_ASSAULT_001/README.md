## Purpose

This packet turns the Codex audit into a bounded emergency debug campaign for
the browser product under `products/noted-host/`. It is written for other
models to reproduce, challenge, and extend the findings. It does not authorize
implementation, deployment, merge, credential use, external probing, or a
security certification.

The emergency trigger is a mismatch between the Lab's strong privacy language
and concrete public browser paths that handle provider credentials and local
user state. Green Nexus/Python gates do not exercise these paths.
