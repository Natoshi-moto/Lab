## Unix-like pipeline stages

```text
intent → map → enrich (FFD) → patch → verify → receipt → human
```

| Stage | Job |
|-------|-----|
| map | inventory, rings, SEC-IDs, STATUS |
| enrich | relevant bloat, anchors |
| patch | one slice |
| verify | allowlisted commands |
| receipt | GitHub/LOOM artifact |
| human | merge / reject |

Hermes executes stages under caps; human owns promotion.
