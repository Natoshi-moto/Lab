## Axiom: hostile machine

**OP-11 / MGR-02:** Treat the **machine** as more hostile than the OS; use that as **reflective defensive posture**.

Implications:

- OS hardening (Tails-like, Qubes-like) reduces blast radius on a **hostile pedestal**.  
- Continuity must not live only on one disk (LOOM multi-path, GitHub, second surface verify).  
- Generative agents run **inside the enemy**; caps + verify + disclosed human.  
- Lineage: Thompson-style trusting-trust concern, updated for archaic AI.
