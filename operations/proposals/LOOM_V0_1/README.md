   argument quality alone, not evidence вЂ” which is the exact failure mode

   reduction.** The design proposes new tooling (`loom verify`, `loom lint`,
   `loom rebuild`, `loom route`, five validators, a content-addressed fence
   format) that does not exist yet. Parking avoids committing build effort
   before either point above is settled.

None of this is a finding against the document's quality. It is an unusually
rigorous, self-critical draft: it states its own author bias up front, names
its single most important unrun experiment, and declines to claim victory
over the system it conflicts with.
