BOOT-SEQUENCE-ACTIVE v1

### 1. Technical summary

Implemented the bounded Sweep 55-R trust polish across Prompt Studio, Canvas, LayerPanel, ScratchDrawer, and project governance docs.

- Prompt Studio pipeline default/fallback copy now uses `Untitled prompt/pipeline`.
- Fresh/default prompt pipelines adopt the typed first-block title when the first draft block is created, and continue adopting first-block renames while the pipeline title is still default or previously synchronized to that first block.
- `Save as prompt` and `Save as block` now title the resulting records from the first-block title first, then the pipeline title, then a safe fallback.
- React Flow default `<Controls />` render/import was removed from Canvas.
- Custom bottom-center `+` / `−` Canvas zoom controls were added.
- React Flow attribution suppression was implemented with supported `proOptions={{ hideAttribution: true }}`.
- Canvas Map/Minimap is collapsed by default, toggleable, and styled with the same compact panel treatment as Layers.
- Layers are collapsed by default when no prior user preference exists.
- ScratchDrawer first-run default height is now 180px, while persisted height and resize behavior remain preserved.
- ScratchDrawer blank banner area toggles open/closed; tab/action controls stop propagation and retain their existing behavior.
- ScratchDrawer banner now includes a centered up/down arrow indicator.
- Docs/governance were updated with the new minimized/collapsed compact-control convention and the new minimap localStorage key.

## FOR HUMAN

### What changed

- Prompt Studio now stops using the stale `Untitled pipeline` wording and uses `Untitled prompt/pipeline` where a prompt-pipeline fallback is needed.
- When you create a fresh prompt/pipeline and type a title into the first block, that title now becomes the saved block title and the fresh/default pipeline title.
- `Save as prompt` and `Save as block` now use the typed first-block title instead of a stale pipeline fallback.
- Canvas no longer renders the default React Flow bottom-left controls. It now has small theme-matching `+` / `−` zoom buttons at bottom center.
- The React Flow attribution is suppressed with the supported React Flow API.
- Canvas Map is collapsed by default and can be toggled; Layers are also collapsed by default on first run.
- ScratchDrawer opens at about half the previous default height, the blank banner toggles open/closed, and a centered arrow now indicates the drawer state.

### Why it matters

These changes remove UI elements that looked like default demo components, fix misleading prompt naming, and make collapsed controls follow a consistent compact workspace style.
