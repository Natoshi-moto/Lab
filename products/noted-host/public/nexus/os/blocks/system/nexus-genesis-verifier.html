Verification is fully local. No data leaves your browser.
Cryptography: Web Crypto API (P-256 ECDSA) for the genesis authority signature  ·  @noble/curves (secp256k1 Schnorr) for the Nostr event signature.
The genesis authority public key is embedded in the genesis output and is the global trust anchor. Any output whose ancestry traces to a genesis signed by a different key is invalid regardless of all other fields.

To independently verify: reconstruct the sigBody as canonical JSON (keys sorted recursively, undefined values dropped), hash with SHA-256, verify the ECDSA-P256 signature against the genesis_authority_pub. Source: NX4 wallet protocol.
