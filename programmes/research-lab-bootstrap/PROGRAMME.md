# Programme: R001 private research-lab bootstrap

## Mission

Create an executable, private, one-repository research facility that can be inspected and challenged from another Claude account without trusting this build narrative.

## Scope

- corpus and authority structure;
- standard-library operator CLI;
- deterministic `CANONICAL_AS_IS` snapshots;
- provider-neutral route packs;
- read-only, hash-bound audit observations;
- local and GitHub Actions verification;
- private GitHub bootstrap command.

## Exclusions

- no claim of complete security;
- no remote GitHub mutation without operator authentication;
- no provider API keys or autonomous model execution;
- no production web application in R001;
- no claim that passing tests establishes the full Nexus thesis.
