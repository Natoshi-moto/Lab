# R016 test notes

The focused campaign covers strict parsing, domain separation, controller-owned
conservation, key rotation, 2-of-3 recovery, terminal revocation, proof of
possession, global key freshness, stale signatures, exact replay, event-ID
collision, paired sibling arrival orders, crash boundaries, lost
acknowledgement, competing writers, capacity, corruption, and caller-anchor
classification.

Passing is bounded evidence for the exercised synthetic profile. It is not
evidence of operational secrecy, physical custody, economic value, network
consensus, physical power-loss durability, or production safety.
